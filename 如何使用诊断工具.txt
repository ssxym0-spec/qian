品类404错误诊断工具使用说明
===================================

遇到品类点击后出现404错误？使用以下工具快速诊断：

方式1：网页版诊断工具（推荐）
--------------------------
1. 确保开发服务器正在运行：
   npm run dev

2. 在浏览器中打开：
   http://localhost:3000/diagnose-categories.html

3. 点击"开始诊断"按钮

4. 查看诊断结果并按建议修复


方式2：命令行测试脚本
--------------------------
直接运行：
   node test-category-mapping.js

这会在命令行中显示详细的诊断信息和修复建议。


需要帮助？
--------------------------
- 快速修复：查看 QUICK_FIX_README.md
- 详细文档：查看 CATEGORY_404_FIX_GUIDE.md
- 修复总结：查看 品类404错误修复总结.md
