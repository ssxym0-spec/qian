## 💡 使用示例

### 基本使用

```tsx
import ScenarioCarousel from '@/app/yunyang/components/ScenarioCarousel';

const scenarios = [
  {
    id: 'gift',
    title: '私房好礼',
    pain_point: '礼物千篇一律？转身即忘',
    solution: '送的不是茶，而是一个完整的故事体系...',
    core_values: [
      { icon: '✨', title: '有面子，显品味' },
      { icon: '💝', title: '绝对走心，独一份' },
      { icon: '📝', title: '记忆点长久' },
    ]
  },
  // 更多场景...
];

export default function MyPage() {
  return (
    <div>
      <h2>场景化应用探索</h2>
      <ScenarioCarousel scenarios={scenarios} />
    </div>
  );
}
```


```tsx
<ScenarioCarousel 
  scenarios={scenarios} 
  autoPlayInterval={5000}  // 5秒
/>
```

### Q2: 如何禁用自动播放？

```tsx
<ScenarioCarousel 
  scenarios={scenarios} 
  autoPlayInterval={999999999}  // 设置超大值
/>
```

### Q3: 如何自定义插图尺寸？

编辑 `ScenarioCarousel.tsx` 中的样式类：
```tsx
className="w-24 h-24 sm:w-32 sm:h-32 md:w-36 md:h-36"
```

### Q4: 如何修改配色？

搜索颜色代码并替换：
- 标题颜色：`text-[#F59E0B]`
- 痛点颜色：`text-red-600`
- 价值勾选：`text-green-600`

### Q5: 数据格式不对怎么办？

组件支持多种格式，会自动兼容：
- 标准格式（推荐）

- 旧的 content 字段格式
