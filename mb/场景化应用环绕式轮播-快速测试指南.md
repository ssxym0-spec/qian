## 🐛 常见问题排查

### 问题1：卡片不会自动播放

**可能原因**：
- 只有一张卡片（少于2张）
- JavaScript错误
- 组件未正确初始化

**排查步骤**：
1. 检查浏览器控制台是否有错误
2. 确认场景数据是否加载成功
3. 刷新页面重试

---

### 问题2：拖拽无响应

**可能原因**：
- 鼠标事件被其他元素拦截
- 浏览器兼容性问题

**排查步骤**：
1. 尝试在不同位置拖拽
2. 更换浏览器测试
3. 检查控制台错误

---

### 问题3：移动端滑动不流畅

**可能原因**：
- 设备性能问题
- 触摸事件冲突

**排查步骤**：
1. 关闭其他应用，释放内存
2. 清除浏览器缓存
3. 在不同设备上测试

---

### 问题4：插图显示异常

**可能原因**：
- 图片URL错误
- 图片加载失败
- 网络问题

**排查步骤**：
1. 检查控制台网络请求
2. 查看是否显示了默认的橙黄色背景
3. 检查图片路径是否正确

---

### 问题5：指示器点击无反应

**可能原因**：
- 点击事件未绑定
- Z-index层级问题

**排查步骤**：
1. 检查控制台错误
2. 尝试点击指示器的不同位置
3. 刷新页面重试

## 🎯 快速验证脚本

在浏览器控制台执行以下代码，可以自动测试部分功能：

```javascript
// 1. 检查组件是否存在

console.log('轮播组件:', carousel ? '✅ 找到' : '❌ 未找到');

// 2. 检查胶囊指示器数量
const indicators = document.querySelectorAll('[aria-label^="切换到场景"]');

// 3. 检查当前激活的指示器
const activeIndicator = document.querySelector('.bg-gradient-to-r.from-orange-400');
console.log('激活指示器:', activeIndicator ? '✅ 找到' : '❌ 未找到');

// 4. 检查插图元素
const illustration = document.querySelector('.w-24.h-24, .w-32.h-32, .w-36.h-36');
console.log('插图元素:', illustration ? '✅ 找到' : '❌ 未找到');

// 5. 模拟点击第二个指示器
if (indicators.length > 1) {
  console.log('模拟点击第2个指示器...');
  indicators[1].click();
  setTimeout(() => {
    console.log('切换完成，请观察卡片是否切换到第2张');
  }, 600);
}

console.log('✅ 测试脚本执行完成');
```
