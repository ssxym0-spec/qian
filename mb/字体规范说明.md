# 网站字体使用规范

## 📝 字体配置

本网站使用 Google Fonts 提供的思源字体系列（中文优化版）：

### 字体来源
- **CDN**：使用国内镜像 `fonts.loli.net` 加速加载
- **字体文件**：
  - 思源宋体（Noto Serif SC）
  - 思源黑体（Noto Sans SC）

## 🎨 字体应用规则

### 1. 主标题 - 思源宋体（Noto Serif SC）

**适用范围**：
- `<h1>` 标签 - 页面主标题
- `<h2>` 标签 - 区块大标题
- `.font-serif` 类 - 需要宋体风格的文本

**特点**：
- ✅ 优雅传统
- ✅ 文化气息浓厚
- ✅ 适合品牌标题
- ✅ 衬线字体，笔画装饰性强

**使用示例**：
```html
<h1 className="font-serif text-4xl font-bold">
  看得见 才安心
</h1>

<h2 className="font-serif text-3xl font-bold">
  四季甄选 · 品类一览
</h2>
```

---

### 2. 副标题 - 思源黑体（Noto Sans SC）

**适用范围**：
- `<h3>` 标签 - 小节标题
- `<h4>` 标签 - 子标题
- `<h5>` 标签 - 次级标题
- `<h6>` 标签 - 最小标题
- `.font-sans-title` 类 - 需要黑体风格的副标题

**特点**：
- ✅ 现代简洁
- ✅ 可读性强
- ✅ 适合信息层级
- ✅ 无衬线字体，清晰易读

**使用示例**：
```html
<h3 className="text-xl font-bold">
  产品特性
</h3>

<h4 className="text-lg font-semibold">
  采摘工艺
</h4>
```

---

### 3. 正文 - 思源黑体（Noto Sans SC）

**适用范围**：
- `<body>` 标签 - 全局默认字体
- `<p>` 标签 - 段落文本
- `.font-sans` 类 - 显式指定黑体

**特点**：
- ✅ 阅读舒适
- ✅ 屏幕显示清晰
- ✅ 适合长文本

**使用示例**：
```html
<p className="text-base text-gray-700">
  我们呈现每一步，只为让您每一口都放心
</p>
```

## 📊 字体层级总览

| 元素 | 字体 | 用途 | 典型大小 |
|------|------|------|----------|
| `<h1>` | 思源宋体 | 页面主标题 | text-4xl ~ text-5xl |
| `<h2>` | 思源宋体 | 区块标题 | text-3xl ~ text-4xl |
| `<h3>` | 思源黑体 | 小节标题 | text-xl ~ text-2xl |
| `<h4>` | 思源黑体 | 子标题 | text-lg ~ text-xl |
| `<h5>` | 思源黑体 | 次级标题 | text-base ~ text-lg |
| `<h6>` | 思源黑体 | 最小标题 | text-sm ~ text-base |
| `<p>` | 思源黑体 | 正文 | text-sm ~ text-base |

## 🎯 使用建议

### 何时使用思源宋体（主标题）

✅ **推荐场景**：
- 品牌 slogan："看得见 才安心"
- 区块大标题："四季甄选 · 品类一览"
- 强调传统文化的标题
- 需要营造高端感的文字

❌ **不推荐场景**：
- 小尺寸文字（<14px）
- 大段正文
- 需要快速扫读的信息

### 何时使用思源黑体（副标题/正文）

✅ **推荐场景**：
- 产品说明、功能介绍
- 卡片标题、列表项
- 所有正文内容
- 需要清晰易读的文字

❌ **不推荐场景**：
- 品牌主视觉标题
- 需要营造文化气息的标题

## 🔧 CSS 实现

```css
/* 全局默认 - 思源黑体 */
body {
  font-family: 'Noto Sans SC', sans-serif;
}

/* 主标题 - 思源宋体 */
h1, h2,
.font-serif {
  font-family: 'Noto Serif SC', serif;
}

/* 副标题 - 思源黑体 */
h3, h4, h5, h6,
.font-sans-title {
  font-family: 'Noto Sans SC', sans-serif;
}
```

---

## 📱 响应式考虑

在移动端，宋体的可读性可能降低，建议：
- h1, h2 在小屏幕上适当减小字号
- 考虑在极小屏幕（<375px）使用黑体替代宋体

## 🌐 字体加载优化

```html
<!-- 预连接 CDN -->
<link rel="preconnect" href="https://fonts.loli.net" />

<!-- 加载字体，使用 display=swap 避免闪烁 -->
<link
  href="https://fonts.loli.net/css2?family=Noto+Sans+SC:wght@400;500;600;700&family=Noto+Serif+SC:wght@400;600;700&display=swap"
  rel="stylesheet"
/>
```

## ✅ 检查清单

使用字体前请确认：

- [ ] 是主标题（h1/h2）？→ 使用思源宋体
- [ ] 是副标题（h3-h6）？→ 使用思源黑体
- [ ] 是正文内容？→ 使用思源黑体（默认）
- [ ] 字号是否合适？（宋体建议 ≥16px）
- [ ] 移动端可读性是否良好？

## 🔄 未来扩展

如需更细粒度控制，可添加以下工具类：

```css
/* 大标题宋体 */
.title-serif {
  font-family: 'Noto Serif SC', serif;
}

/* 副标题黑体 */
.subtitle-sans {
  font-family: 'Noto Sans SC', sans-serif;
}

/* 正文黑体（显式） */
.body-sans {
  font-family: 'Noto Sans SC', sans-serif;
}
```
