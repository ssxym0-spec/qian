## 📊 后端 API 要求

### 接口地址
```
GET http://localhost:3000/api/public/landing-page
```

### 返回数据格式
参考 `test-data-example.json` 文件中的完整示例。

关键字段：
- `plot` - 地块信息
  - `name` - 地块名称
  - `carousel_images` - 轮播图片数组
  - `info_list` - 信息列表数组
  - `value_summary` - 价值总结
- `categories` - 品类数组（4个）
  - `name` - 品类名称
  - `image_url` - 图片URL
  - `description` - 描述
  - `percentage` - 产量占比
  - `harvest_period` - 采摘期
- `cta_bg` - CTA背景图片URL
