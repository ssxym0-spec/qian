# 场景轮播 - 快速验证指南 ✅

## 🚀 10秒快速验证

### 步骤1: 启动
```bash
npm run dev
```

### 步骤2: 访问
```
http://localhost:3000/yunyang
```

### 步骤3: 查看
1. 切换到"私人定制"或"企业领养"标签
2. 滚动到"场景化应用探索"区块
3. 应该看到这样的布局：

```
桌面端:
┌──────────────────────────────────────┐
│  ←  [左侧内容]  [右侧图片]  →        │
│      ━━━━━ ━ ━                       │
└──────────────────────────────────────┘

移动端:
┌──────────────┐
│  [上方内容]  │
│  [下方图片]  │
│   ━━━━ ━ ━   │
└──────────────┘
```

## 🐛 常见问题

### Q: 看不到左右箭头？
**A**: 正常！移动端会隐藏箭头，使用拖拽切换。

### Q: 没有背景图片？
**A**: 正常！会显示渐变色占位符 + 淡化的场景图标。

### Q: 布局是上下分栏？
**A**: 检查浏览器窗口宽度，小于768px会自动切换为上下布局。

### Q: 拖拽不灵敏？
**A**: 需要拖动超过100px才会切换，否则会回弹。
