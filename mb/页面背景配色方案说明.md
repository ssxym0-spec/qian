# 页面背景配色方案说明

## 🎨 设计理念

基于茶文化和页面功能，为每个页面赋予独特的主题色，营造和谐统一的视觉体验。

---

## 📊 四大页面配色方案

### 1️⃣ 首页 - 大地暖色系（琥珀金）

**背景设置**：
```tsx
bg-gradient-to-b from-amber-50 to-white
```

**颜色值**：`#fffbeb` (amber-50) → `#ffffff` (white)

**设计理念**：
- 🏡 **欢迎页基调**：温暖的琥珀色营造亲切感，欢迎用户进入茶园
- 🍵 **茶汤金黄**：琥珀色象征优质茶汤的金黄色泽
- ☀️ **阳光温润**：如同阳光洒在茶园的温暖感受
- 🎨 **淡雅不抢眼**：使用 -50 级别的淡色，不干扰内容阅读

**适用文件**：`app/page.tsx`

---

### 2️⃣ 溯源页 - 茶青绿色系（翡翠绿）

**背景设置**：
```tsx

```

**颜色值**：`#ecfdf5` (emerald-50) → `#ffffff` (white)

**设计理念**：
- 🌿 **茶叶本色**：翡翠绿象征新鲜茶叶的青翠色
- 🔍 **制作过程**：溯源关注从鲜叶到成品的全过程
- 💎 **珍贵品质**：翡翠的珍贵感与优质茶叶品质相呼应
- 🌱 **生机活力**：比灰色更有生机，传达茶叶的鲜活感

**适用文件**：
- `app/suyuan/[category]/page.tsx`（批次列表页主背景）
- `app/suyuan/components/CategoryFilter.tsx`（批次列表页粘性顶部导航栏）

- `app/suyuan/components/BatchDetailTopNav.tsx`（批次详情页固定顶部导航栏）

## 🎯 配色原则

### 1. 统一性原则
- **渐变方向**：所有页面都采用 `from-xxx-50 to-white` 的从上到下渐变
- **色阶统一**：都使用 `-50` 级别的淡色（Tailwind 最浅的色阶之一）
- **结束色统一**：都渐变到纯白色 `white`

### 2. 区分性原则
- **不同色系**：四个页面使用四种不同的颜色系列
  - 首页：amber（琥珀）
  - 溯源：emerald（翡翠绿）
  - 认养：stone（石色）
  - 生长：green（嫩绿）
- **视觉身份**：用户可以通过背景色快速识别当前页面

### 3. 文化契合原则
- **茶文化相关**：每种颜色都与茶文化、茶园、茶叶有关联
- **功能呼应**：颜色选择与页面功能相呼应
- **情感传达**：通过颜色传达相应的情感和氛围

### 4. 用户体验原则
- **不抢焦点**：淡雅的背景不干扰内容阅读
- **视觉舒适**：柔和的渐变减轻视觉疲劳
- **层次分明**：渐变从顶部到底部，自然引导视线

## 💡 未来优化建议

### 可选优化方向
1. **季节主题**：根据季节（春夏秋冬）调整背景色调
2. **时段变化**：白天/夜晚自动切换配色方案
3. **交互反馈**：页面切换时的背景色过渡动画
4. **个性化**：允许用户自定义偏好的背景主题

### 备选方案
如果未来需要调整，可以考虑：
- **更深渐变**：从 `-100` 级别开始，增强对比
- **双色渐变**：从一个颜色渐变到另一个颜色（如 `from-amber-50 to-green-50`）
- **径向渐变**：使用 `bg-radial-gradient` 创建中心辐射效果
