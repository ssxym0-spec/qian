## 🎨 布局区域说明

### 1. 场景标题区 (左上)
**桌面端**:
- 白色背景卡片
- 圆角 + 浅阴影
- 大号图标 + 标题
- 占据左上角1个网格单元

**内容**:
- Emoji图标 (60-70px)
- 场景名称 (30-40px 粗体)

### 2. 背景图片显示区 (右侧)
**桌面端**:
- 占据右侧整个高度 (`row-span-2`)
- 最小高度 400px
- 圆角 + 阴影

**有图片时**:
- 显示实际背景图
- `object-cover` 填充

**无图片时**:
- 橙黄色渐变背景
- 虚线边框
- 淡化的场景图标
- "背景图片显示区" 文字提示

### 3. 痛点文案区 (左下)
**桌面端**:
- 白色背景卡片
- 圆角 + 浅阴影
- 占据左下角1个网格单元

**内容结构**:
1. **客户困扰** (小标题)
   - 灰色、大写、字间距
   
2. **痛点描述** (正文)
   - 较大字号、行高适中

3. **💡 定制方案** (高亮卡片)
   - 橙色渐变背景
   - 左侧橙色边框
   - 加粗文字

### 4. 价值文案区 (底部)
**桌面端**:
- 横跨两列 (`col-span-2`)
- 白色背景卡片
- 圆角 + 浅阴影

**内容布局**:
- ✨ 标题
- 3列网格布局 (lg屏幕)
- 每个价值点：
  - 图标 + 标题 + 描述
  - 渐变背景卡片
  - Hover悬停效果

## 🎯 CSS Grid 技术细节

### 桌面端网格配置
```css
grid-cols-2          /* 2列布局 */
gap-6 lg:gap-8       /* 间距 24-32px */
```

### 区域跨度
```css
/* 场景标题区 */
默认 (1列1行)

/* 背景图片区 */
row-span-2          /* 跨2行 */

/* 痛点文案区 */
默认 (1列1行)

/* 价值文案区 */
col-span-2          /* 跨2列 */
```

### 价值区网格
```css
grid-cols-1 lg:grid-cols-3   /* 1列或3列 */
gap-4                        /* 间距 16px */
```

## ✨ 视觉细节

### 卡片样式
- **背景**: 白色 `bg-white`
- **圆角**: `rounded-xl` (12px)
- **阴影**: `shadow-sm` (浅阴影)
- **边框**: `border border-stone-100` (淡灰色)

### 定制方案卡片
- **背景**: 橙色渐变 `from-orange-50 to-amber-50`
- **边框**: 左侧4px橙色 `border-l-4 border-[#F59E0B]`
- **内边距**: `p-5`

### 价值卡片
- **背景**: 渐变 `from-stone-50 to-white`
- **边框**: `border-stone-200`
- **Hover**: `hover:shadow-md` 悬停阴影

## 📱 响应式策略

### 桌面端 (≥768px)
- 使用CSS Grid 2列布局
- 显示左右箭头
- 价值区3列显示

### 移动端 (<768px)
- 垂直堆叠布局
- 隐藏左右箭头
- 价值区1列显示
- 触摸滑动切换

## 🎭 交互功能

### 轮播控制
- **左右箭头** (桌面端)
  - 圆形白色按钮
  - 浮在轮播外侧
  - Hover放大效果

- **拖拽/滑动**
  - 支持鼠标拖拽
  - 支持触摸滑动
  - 阈值：100px

- **底部指示器**
  - 胶囊式设计
  - 当前项橙色加长
  - 点击跳转

### 切换动画
- **过渡时间**: 500ms
- **缓动函数**: ease-out
- **transform**: translateX

## 📝 技术要点

### CSS Grid关键类名
```css
/* 容器 */
grid grid-cols-2 gap-6

/* 图片区跨行 */
row-span-2

/* 价值区跨列 */
col-span-2

/* 价值内部网格 */
grid grid-cols-1 lg:grid-cols-3
```

### 响应式隐藏/显示
```css
/* 移动端显示 */
md:hidden

/* 桌面端显示 */
hidden md:grid
```
