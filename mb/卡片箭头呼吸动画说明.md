# 卡片箭头呼吸动画说明

## 🎯 功能概述

为所有可点击卡片的右箭头添加呼吸动画效果，引导用户点击，提升交互体验。

## 📋 应用范围

所有箭头已统一样式并添加呼吸效果：

| 卡片类型 | 位置 | 箭头颜色 | 组件文件 |
|---------|------|---------|---------|
| **品类信息卡片** | 采摘期同一行右侧 | 金色 `text-grain-rain-gold` | `app/page.tsx` |
| **每日日志卡片** | 底部右侧（记录人信息旁） | 橙色 `text-orange-500` | `app/components/growth/DailyLogCard.tsx` |
| **溯源批次卡片** | 底部右侧（制茶师信息旁） | 琥珀色 `text-amber-600` | `app/suyuan/components/BatchCard.tsx` |

## 🎨 动画效果

### 呼吸动画参数

```css
@keyframes arrow-breathe {
  0%, 100% {
    opacity: 0.6;          /* 半透明 */
    transform: translateX(0);  /* 原位 */
  }
  50% {
    opacity: 1;            /* 完全不透明 */
    transform: translateX(4px);  /* 向右移动4px */
  }
}

.arrow-breathe {
  animation: arrow-breathe 2s ease-in-out infinite;
}
```

### 动画特点

- ✨ **透明度变化**：0.6 ⇄ 1.0（60% ~ 100%）
- 🏃 **位置移动**：向右移动 4px 再回到原位
- ⏱️ **动画周期**：2秒一个循环
- 🌊 **缓动函数**：ease-in-out（平滑进出）
- 🔄 **无限循环**：持续吸引用户注意

## 📐 统一规范

### 箭头样式统一

**之前**：
- 每日卡片使用文字箭头 `→`
- 其他卡片使用 SVG 箭头
- 样式不统一

**现在**：
- ✅ 所有卡片统一使用 SVG 箭头
- ✅ 统一尺寸：`w-5 h-5`（20x20px）
- ✅ 统一动画：`arrow-breathe` 类
- ✅ 各自保持品牌色彩

### SVG 箭头代码

```html
<svg 
  className="w-5 h-5 text-[颜色] arrow-breathe" 
  fill="none" 
  stroke="currentColor" 
  viewBox="0 0 24 24"
>
  <path 
    strokeLinecap="round" 
    strokeLinejoin="round" 
    strokeWidth={2} 
    d="M9 5l7 7-7 7" 
  />
</svg>
```

## 💡 设计理念

### 为什么要呼吸动画？

1. **引导性** 👉
   - 明确告诉用户"这里可以点击"
   - 向右移动暗示"进入下一级"

2. **吸引力** ✨
   - 动态元素自然吸引视线
   - 适度闪烁增加关注度

3. **品质感** 🎨
   - 精致的动画提升产品质感
   - 细节处理体现专业性

4. **温和性** 😌
   - 2秒周期不急不缓
   - 半透明起始避免过于刺眼

### 为什么统一使用 SVG？

1. **缩放清晰** 📐
   - 矢量图形，任何尺寸都清晰
   - 响应式设计友好

2. **样式灵活** 🎨
   - 可以自由改变颜色
   - 可以添加动画效果

3. **语义明确** 📝
   - SVG 图标语义清晰
   - 可访问性更好

4. **性能优异** ⚡
   - 文件体积小
   - 渲染性能好
