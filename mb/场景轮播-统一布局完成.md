# 场景轮播 - 统一布局完成 ✅

## 📐 布局结构（桌面端 & 移动端统一）

### 布局顺序
1. **标题区** - 场景图标 + 名称
2. **痛点区** - 客户困扰
3. **插画图** - 背景图片或占位符
4. **方案区** - 💡 定制方案（橙色高亮）
5. **价值区** - ✨ 核心价值

---

```
┌─────────────────┬─────────────────┐
│   1. 标题区     │                 │
│                 │                 │
├─────────────────┤   3. 插画图     │
│                 │                 │
│   2. 痛点区     │  (row-span-2)   │
│                 │                 │
├─────────────────┴─────────────────┤
│                                   │
│        4. 方案区 (col-span-2)      │
│                                   │
├───────────────────────────────────┤
│                                   │
│        5. 价值区 (col-span-2)      │
│                                   │
└───────────────────────────────────┘
```

### 技术实现
- **网格系统**: `grid grid-cols-2`
- **插画图跨行**: `row-span-2` (占据右侧全高)
- **方案区横跨**: `col-span-2`
- **价值区横跨**: `col-span-2`

```
│   2. 痛点区       │
├───────────────────┤
│   3. 插画图       │
├───────────────────┤
│   4. 方案区       │
├───────────────────┤
│   5. 价值区       │
└───────────────────┘
```

### 技术实现
- **垂直堆叠**: `space-y-5`
- **顺序一致**: 与桌面端完全相同的内容顺序
- **卡片化**: 每个区域都是独立的白色卡片

---

## 🎨 区域详细说明

### 1️⃣ 标题区
**桌面端**:
- 图标 60-70px
- 标题 30-40px
- 白色卡片背景

**移动端**:
- 图标 48px
- 标题 24px
- 白色卡片背景

### 2️⃣ 痛点区
**桌面端**:
- "客户困扰" 小标题
- 痛点文案 (18px)
- 白色卡片背景

**移动端**:
- "客户困扰" 小标题
- 痛点文案 (16px)
- 白色卡片背景

### 3️⃣ 插画图
**桌面端**:
- 占据右侧全高 (row-span-2)
- 最小高度 350px
- 有图显示图片，无图显示占位符

**移动端**:
- 固定高度 256px (h-64)
- 有图显示图片，无图显示占位符

### 4️⃣ 方案区
**桌面端**:
- 横跨整个宽度 (col-span-2)
- 橙色渐变背景
- 左侧橙色边框
- 大号图标和标题

**移动端**:
- 白色卡片包裹
- 内部橙色渐变背景
- 左侧橙色边框

### 5️⃣ 价值区
**桌面端**:
- 横跨整个宽度 (col-span-2)
- 3列网格展示核心价值
- 每个价值卡片有渐变背景

**移动端**:
- 垂直堆叠展示核心价值
- 每个价值卡片有渐变背景

## 🎨 颜色方案

### 主色调
- **橙色**: `#F59E0B` (方案区边框、标题)
- **石色**: `stone-*` (文字、边框)
- **白色**: `white` (卡片背景)

- **容器背景**: `from-white to-stone-50`
- **方案卡片**: `from-orange-50 to-amber-50`
- **插画占位**: `from-orange-100 via-amber-100 to-yellow-100`
- **价值卡片**: `from-stone-50 to-white`
